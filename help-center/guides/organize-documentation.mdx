## Overview

Organize your documentation in a few easy steps:

<Steps>
  <Step title="Create a project">
    Click the "New Project" button from your dashboard and name your project.
  </Step>
  <Step title="Add sections">
    Use the "Add Section" feature to structure documentation (e.g., Setup, Usage).
  </Step>
  <Step title="Write content">
    Add markdown or rich text for each section. Save changes as you go.
  </Step>
</Steps>
